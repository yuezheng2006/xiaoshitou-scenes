# 小石头 IP

品牌语境见 `thunderstone-brand.md`（消歧参考，非官方声明）。

## 核心定位

**小石头 (Little Stone)** — Thunderstone 雷石正文配图角色：flat 2D 简笔、物理动作、短标签、大留白。

```text
形体：圆角胶囊 / 竖椭圆，#f39200 雷石橙实心体
眼睛：白色小圆 + 可选 tiny 瞳孔（不要 Logo 护目镜、不要 anime 大眼）
肢体：细黑线；表情靠动作，汗滴 0–1 个即可
身上：禁止任何 Logo /「雷石」/ 品牌印花
```

**角色锚点（唯一）**：`assets/brand/thunderstone-xiaoren-ref.png`

`thunderstone-exhibition-brand-ref.png` 可用于展会/背景气质参考，**不是**小石头五官规范。Logo 几何以 `thunderstone-logo-core.png` 为准。

## 是谁

略萌、认真、略笨、有点倔——靠 **物理动作** 表达，不堆五官。

| 英文名 | **Little Stone** |

## 默认形象

### 身体

- 品牌橙 `#f39200` 实心体。
- 轮廓：简洁 **圆角胶囊 / 竖椭圆**，高宽比约 **1.2:1 ~ 1.5:1**。
- 可有极细黑描边；同批统一。
- 同图可多小人、略拉长/略压扁——**仍是小石头**，不要变成另一种 mascot。
- **全身纯色 #f39200**，禁止 Logo /「雷石」/ 任何品牌印花。

### 眼睛

- **两个白色小圆眼**，可带 **tiny 黑色瞳孔点**；不要 Logo 叠眼、不要大护目镜、不要 anime 眼。
- 表情优先靠 **动作、倾斜、汗滴（0–1 个）**；眉毛 **可选**，最多 **一条** 简单短线。

### 肢体

- 细线胳膊、细线腿；认真做物理动作。

### 性格（靠动作）

- 动作 **认真、较劲**（拉、挡、解结、改字）。
- 偶尔 **汗滴 / 身体倾斜** 表达吃力或懵。
- 标签话术偏 **打工人真实口语**（对齐一下、谁改的、先排查）。

## 品牌标记位置

- **禁止** 在小石头身上 / 旁用「雷石」箭头引导。
- Logo /「雷石」**仅**出现在工牌等道具牌面（见 `thunderstone-brand.md`）。

## 同类场景一致性

一次生成 **2 张及以上** 时，Character Lock：

| 全批一致 | 可变 |
| --- | --- |
| `#f39200` 橙身 | 姿势 / 动作 |
| 胶囊轮廓（对照 ref） | 0–1 汗滴 |
| 白色小圆眼 + 可选 tiny 瞳孔 | 简单短线眉（可选） |
| flat 2D 简笔 | 场景道具 |
| 细线肢体 | 标签 |

**QA**：对照 `thunderstone-xiaoren-ref.png`——同一 **小石头**，不是新物种。

## 场景发散

- 办公/权限：可加工牌——叙事道具；Logo 只在牌面。
- 物件：简笔 + 少量真实感；**不要** 3D / 写实摄影 / 海外商业风。

## 禁止

- 脱离 ref 形体（Logo 导出体、星火体、麦克风人、桌游风）。
- 把 Logo 眼型 **套到小石头脸上**。
- 复杂眉眼、3D、写实摄影、海外精致插画。
- 自画 Thunderstone Logo（须用 `thunderstone-logo-core.png`）。
- 小石头 **身上** 出现 Logo /「雷石」/ 品牌胸标。
- 同批变成不同 mascot 或不同画法。

## 必须承担核心动作

删掉小石头后隐喻仍成立 = 失败。
