# QA Checklist

## 必过项

### 母版与构图

- 标准模式已锁定 01–06 母版类型；PNG 有则比对，无则用 `master-selection.md` 骨架。
- 16:9 纯白 `#FFFFFF`；彩蛋长卷近白；留白足够。
- 场景覆盖约宽 60–72%、高 38–50%；3 秒读懂一个物理冲突。
- 无元素清单化、无母版换皮复刻。

### 简笔现场

- flat 2D 简笔；**不要** 3D、写实摄影、海外商业风。
- 物件与小石头同一简笔语言；无 UI 截图。

### 小石头 IP

- 形体对照 `thunderstone-xiaoren-ref.png`，皮肤 `#f39200`。
- 白圆眼 + 可选 tiny 瞳孔；**不要** Logo 护目镜眼、不要 anime 眼。
- 表情靠 **动作**，汗滴 0–1 个即可。
- **身上** 无 Logo /「雷石」/ 品牌标记；非办公场景不加工牌。
- 2 张及以上：全批同一小石头 + flat 2D。

### 品牌

- 品牌语境消歧（见 brand 文档）；不是 Quest 桌游、不是雷石投资私募视觉。
- 官方 Logo 几何须与 `thunderstone-logo-core.png` 一致。

## 失败信号

- 不像小石头 ref（Logo 眼、星火体、新 mascot 物种）。
- 过度细节、3D、写实、海外 editorial 风。
- 小石头只站看、不承担动作。
- 同批形体/画法不一致。
- 身上误印 Logo /「雷石」；工牌 Logo 与「雷石」异色；旁有「雷石」引导标签。
- 堆很多五官细节，与物理动作脱节。

## 交付判断

1. 第一眼：简、干净。
2. 第二眼：看懂物件与小石头的物理冲突。
3. 标签：共鸣。

不合格 = 重生成，不要交付。
