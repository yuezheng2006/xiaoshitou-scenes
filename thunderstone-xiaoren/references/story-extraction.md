# 故事提炼

## 目标

把抽象内容变成一个“真实物件现场”。先找处境，再找物件。

## 五步提炼法

### 1. 找读者处境

问：

- 这段内容里，读者是谁？
- 他正在被什么困住？
- 他最熟悉的那一句话是什么？
- 他看到图后应该说“这不就是我吗”还是“这个观点好清楚”？

常见读者 / 品牌叙事处境：

- 互联网打工人：会议、消息、需求、周报、下班失败。
- 程序员：报警、回滚、Review、排查、上线前后。
- 产品/设计：对齐、改稿、评审、优先级、需求漂移。
- AI 时代职场人：岗位重组、简历筛选、自动化替代、能力过期。
- 创作者：选题焦虑、素材堆积、内容复用、发布拖延。
- 一人公司：获客、交付、知识库、产品化、复盘。

### 品牌通用处境

- 产品发布：功能亮点、用户价值、场景演示。
- 行业观察：KTV、家庭娱乐、音乐科技趋势（可选，不强制）。
- 团队故事：项目节点、产品演化、服务升级。
- 用户场景：点歌、聚会、家庭 K 歌、门店体验、曲库更新、包厢设备。

优先把 K 歌 / 产品现场变成物理动作，例如：

- 「曲库还没更完，客人先点了」→ 纸条涌出，小石头用挡板推回。
- 「切歌切错了」→ 放大镜下改歌单，小石头在改。
- 「手机点歌连不上」→ 线缆打结，小石头在解结。


### 2. 找核心冲突

一句话压缩：

```text
谁，被什么，以什么方式，推向什么结果？
```

例：

- 打工人被会议卡片拉进电脑。
- 打工人被消息纸条淹没。
- 程序员被一根打结的报警线缆困住。
- 简历还没见到人，就先被筛子筛了一遍。

### 3. 把抽象词变成物理动作

优先选能一眼看懂的动作：

- 拉进、拽回、压住、筛掉、盖章、卡住、打结、贴标签、塞入、抽出、挡住、托住、剪断、缝上、称重、钉住、夹住。

不要只画抽象名词：

- 不要画“焦虑”。
- 画“纸条不断飞出来，小石头拿纸板挡”。
- 不要画“自动化替代”。
- 画“工牌被贴上自动化标签，小石头拉住挂绳”。

### 4. 找真实物件

物件必须和读者经验相连。

好物件：

- 电脑、手机、键盘、线缆、工牌、放大镜、筛子、印章、便签、长尾夹、沙漏、咖啡杯、台灯、秤、纸堆、文件夹。

不要为了高级而找陌生物件。物件越日常，荒诞动作越容易成立。

### 5. 写短标签

短标签来自真实语境，不是概念总结。

优先：

- “对齐一下”
- “马上回”
- “再催一下”
- “谁改的”
- “先回滚”
- “小问题”
- “再改改”
- “AI 先做”
- “我还在吗”
- “先筛一遍”
- “关键词”
- “人呢”

避免：

- “提升协作效率”
- “信息过载管理”
- “多维度能力升级”
- “用户价值闭环”
- “系统化方法论”


## 选题优先级

优先画：

- 有真实痛感的高频瞬间。
- 能被一个物理动作表达的抽象观点。
- 有传播钩子的用户处境。
- 可以靠 2-4 个短词点破的场景。

少画：

- 需要长解释才懂的复杂系统。
- 没有具体读者处境的抽象理念。
- 只有“好看物件”但没有冲突的摆拍。

## 彩蛋长卷提炼法

当用户要求“彩蛋模式 / 长卷故事图 / 超横版 / 个人经历 / 项目复盘 / 产品演化 / 成长路径”时，不要按单张正文图提炼。先把内容拆成一条连续故事线。

### 1. 找主线

一句话写清这张长卷讲什么：

```text
从 {起点身份/旧处境}，经过 {关键转折}，走到 {现在关注/新方向}。
```

### 2. 拆 5-8 个节点

默认 5-7 个节点最稳，内容很丰富时才到 8 个。

节点格式：

```text
节点主题：
真实物件：
小石头动作：
节点注释：
为什么它是这一段的代表：
```

彩蛋长卷里的节点不要编号。不要写“1/2/3”、圆形序号、步骤号或里程碑编号；让路线、真实物件和手写短注释自然表达先后关系。

### 3. 左起右收

左侧不是大标题，而是身份开场：

```text
Hello!
我是 {名字/身份}
{一句主线}
{起点信息}
```

右侧不是联系方式，而是当前关注或结论：

```text
我现在关心的是
{关注1}
{关注2}
{关注3}
```

### 4. 物件必须像记忆点

长卷里的物件不是装饰，也不是每个节点都随便找一个图标。每个物件都要像一个阶段的“实物锚点”：

- 做设计：尺子、色卡、作品集、奖章。
- 做内容：稿纸、相机、计数器、笔记本。
- 做产品：积木、钥匙、标牌、工具箱。
- 做复盘：放大镜、文件夹、线轴、便签堆。
- 做成长路径：路标、票据、书、旧工牌、新标签。

### 5. 保留母版骨架

`07` 长卷类型是彩蛋模式的核心骨架（PNG 缺失时用 `style-dna.md` + 本节）。提炼时先对齐：

- 左侧开场。
- 一条细路线。
- 节点高低起伏要自然、不规律；不要把节点排成紧凑平直的一排，也不要做成规律正弦波。
- 节点之间要留出大块干净空白，并且间距不完全相等，让长卷像被慢慢展开，而不是被塞满或被网格化。
- 路线可以有平缓段、突然抬高、浅弯、深一点的下探和安静收尾；不要按“上、下、上、下”机械交替。
- 小石头逐段参与；每节点同一小石头 + 物理动作。
- 真实物件承载阶段记忆。
- 节点文案像写在空白里的手写旁白，默认不放进便利贴、纸条、卡片或标签框。
- 右侧收束。

然后替换当前内容。不要把母版里的**他人经历、第三方平台、样例品牌名**当成默认内容；只有用户提供等价事实时才可写入。
