# 使用示例

## 生图入口（Codex）

安装 Skill 后，直接用中文说意图即可触发。推荐明确写：

- `小石头实物图：...`
- `小石头手绘图：...`

不必写配色、角色细节，也不必写 `Use $little-stone-scenes`。

```bash
# 安装（一次性）
cp -R ./little-stone-scenes "${CODEX_HOME:-$HOME/.codex}/skills/"
```

## 常用 Prompt

### 先规划，暂不生图

```text
先不要生图。请分析下面这篇中文内容，分别给出小石头实物图和小石头手绘图的配图方案建议。

<粘贴内容>
```

### 生成一组实物图

```text
小石头实物图：把下面这篇内容生成 4 张 16:9 正文配图。

<粘贴内容>
```

### 生成一张手绘图

```text
小石头手绘图：把下面这个 Agent 工作流解释成一张 16:9 白板图。

<粘贴流程或说明>
```

### 单个观点一张实物图

```text
小石头实物图：为这个观点生成一张 16:9 正文配图：

产品功能很多，但用户真正记住的往往只有一个瞬间。
```

### 方法论一张手绘图

```text
小石头手绘图：把这个方法论画成一张结构清楚但不 PPT 的解释图：

先收集素材，再沉淀成可复用模块，最后组合成稳定输出。
```

### 知识卡

```text
小石头知识卡：把这套『四步内容复用方法』做成一张 3:4 收藏图，安排 2-3 个小石头在步骤里分工协作。

<粘贴方法内容>
```

### PPT 演讲模式

```text
PPT 演讲模式：主题《AI 配图工作流是怎么搭起来的》，10 页以内。先出整套导演规划卡，等我确认后再出 1-2 页样张。

<粘贴大纲或逐字稿>
```

### 长卷故事图

```text
小石头实物图：把这个项目复盘做成一张超横版长卷故事图。

主题：从一个本地想法，慢慢变成可规模化的产品。
节点：起点、原型、试点、反馈、规模化、新场景、持续迭代。
```

### 聚会娱乐 / 抢麦合唱

```text
小石头实物图：生成一张 16:9 配图，表达朋友聚会 K 歌时正在接麦合唱。画面用双麦克风、接麦号码牌、铃鼓和彩纸点做主物件；两个小石头一高一矮，一个递麦，一个接麦，姿态和体型要明显不同，不要像复制。
标签：轮到你 / 接麦 / 合唱吧
```

### 家庭娱乐 / 客厅 K 歌

```text
小石头实物图：生成一张 16:9 配图，表达家庭客厅 K 歌音箱已经准备开唱。画面用圆润家用 K 歌音箱、短柄现代无线麦、电视遥控器、歌词牌、沙发靠垫和零食盘做主物件，小石头正在把遥控器拖向歌词牌，另一只手扶住 K 歌音箱。
标签：客厅开唱 / 音箱开了 / 先点歌
```

### 车载 K 歌 / 路上开麦

```text
小石头实物图：生成一张 16:9 配图，表达车载 K 歌和出行娱乐。画面用车钥匙、路线票、车载小音箱、短柄现代无线麦和旅行歌单卡做主物件，小石头把歌单从路线图上拖向车载麦克风。
```

### 曲库与内容更新


### 员工工牌系列

```text
小石头实物图：生成一张 16:9 配图，画员工服务工牌和值班牌作为主物件，工牌作为身份叙事道具，小石头正在把「值班中」贴纸贴到工牌上。
标签：值班中 / 去现场 / 先处理
```

```text
小石头实物图：生成一张 16:9 配图，画现场服务权限牌、服务工单和麦克风检查卡，小石头拉住挂绳准备去处理包房呼叫。
标签：有呼叫 / 刷卡进场 / 马上到
```

> 如需在工牌牌面加入真实品牌 Logo，请显式说明并自行确认授权；默认示例不展示具体品牌。

### 作者出镜

```text
小石头实物图：老杨和小石头一起整理一堆 prompt 草稿，生成一张 16:9 配图。
标签：先试 / 再改 / 留一版
```

```text
小石头手绘图：作者 yuezheng2006 在白板旁解释双模式路由，小石头负责拉线连接实物图和手绘图。

```

### 改图

```text
这张小石头手绘图方向对，但文字太多。请减到 4 个短批注，其他保持不变。
```

```text
这张小石头实物图方向对，但主物件太重。请保持隐喻不变，把画面变轻一些。
```

## 试跑场景

18 个实物图试跑场景（含多人场景，其中 15 个已有样张）的**一图一摘要 + 可复制 prompt**，见 [test-scenarios.md](test-scenarios.md)。

示例：

```text
小石头实物图：生成一张 16:9 配图：三个 Agent 接力传任务。多人场景，2-3 个同 IP 小石头（共享 #f39200 橙身、白圆眼，但体型、姿态、朝向、分工明显不同，不要复制粘贴）。
标签：Agent A / 传给 B / 接着干
```

```text
小石头手绘图：生成一张 16:9 白板图：解释 RAG 如何先检索文档，再把相关片段交给模型回答。
批注：先检索 / 取相关段 / 再回答
```
