**预期结果**：
- ✅ 所有 Tier 1 通过 → CONFIRMED 可交付

### 场景 3: 容忍度边界测试

**测试目标**：验证容忍度是否正确生效

**测试点**：
```
1. 臂锚点位置：约上 1/4（27%）
   - 旧标准：必须上 1/3（33%）→ FAIL
   - 新容忍度：上 1/4-2/5（25-40%）→ PASS

2. 胶囊宽高比：约 1:1.28
   - 旧标准：1:1.3-1:1.6 → FAIL
   - 新容忍度：1:1.2-1:1.7（±0.1）→ PASS

3. 眼睛直径：约 15.5% 体宽
   - 旧标准：12-18% → PASS
   - 新容忍度：10-20%（±2%）→ PASS
```

**Tier 1 检查清单**：
- [ ] **L1 计数**：2 臂 + 2 腿 ✓
- [ ] **E1 眼睛**：两只白圆眼 ✓
- [ ] **E2 无嘴**：胶囊上无嘴线 ✓

**Tier 2 观察点**（不阻断交付）：
- [ ] L2 臂锚点：略低于标准 1/3，但在容忍范围内（27%）
- [ ] E4 胶囊比例：略低于标准，但在容忍范围内（1:1.28）

**预期结果**：
- ✅ Tier 1: PASS（所有核心检查通过）
- ⚠️ Tier 2: 可标注"臂锚点略低于标准位置，在可接受范围内"
- 交付判断：**CONFIRMED 可交付**（不因边界值被阻断）
