# Confirm Gate 简化方案（P1.1）

日期：2026-07-16  
状态：准备执行  
目标：将 20+ 项 CRITICAL 检查简化为分级系统

## 背景问题

**当前状态**：
- 小石头检查项：6 项（L1-L4 + E1-E2）
- 老杨检查项：7 项（P1-P7）
- 跨图检查项：7 项（X0-X6）
- **总计 20 项全部标记为 CRITICAL**

**问题**：
1. Agent 检查负担重（每张图 20+ 项）
2. 失败率高（任一 CRITICAL FAIL → 整张返修）
3. 标准主观（"长但不细"如何量化？）
4. 过度精确（"上 1/3" 边界模糊）

## P1.1 目标

**简化为三级检查系统**：
```text
Tier 1: CRITICAL（3-5 项）— 绝对不能错
Tier 2: IMPORTANT（5-7 项）— 尽量满足
Tier 3: NICE-TO-HAVE（其余）— 建议但不强制
```

**原则**：
- 只有真正影响 IP 识别的才是 CRITICAL
- 细节优化降级为 IMPORTANT / NICE-TO-HAVE
- 引入容忍度（量化可接受范围）

## 分级方案

### Tier 1: CRITICAL（必须通过）

**小石头**（3 项）：
```text
L1 计数：每体 2 臂 + 2 腿
E1 眼睛：两只白圆眼
E2 无嘴：胶囊上无嘴线
```

**老杨**（3 项）：
```text
P1 眼镜：大镜片浅灰细框（第一识别）
P3 脸型：长但不细长方椭圆（宽度可见）
P6 可见性：正面或 3/4 可见上述锚点
```

**跨图**（2 项）：
```text
X1 小石头形体：批内宽高比一致（±10%）
X4 老杨眼镜发型：批内一致（无丢失）
```

**交付判断**：
- 所有 Tier 1 通过 → 可交付
- 任一 Tier 1 失败 → 必须返修

### Tier 2: IMPORTANT（尽量满足）

**小石头**（3 项）：
```text
L2 锚点：臂从体侧上 1/3 ± 10%、腿从底缘
L3 线质：细黑线连续、非 3D 圆管
L4 末端：每臂 ≤1 个简化握持
```

**老杨**（4 项）：
```text
P2 发型：短直发 3-5cm（±1cm 可接受）
P4 嘴唇：厚下唇（识别增强）
P5 穿搭：米色 T + 卡其短裤（默认）
P7 表情：平静/专注（不夸张）
```

**跨图**（3 项）：
```text
X2 小石头眼睛：批内眼距/大小一致
X5 老杨年龄肤色：35-40 岁范围稳定
X6 老杨嘴唇体态：批内厚度/比例一致
```

**交付判断**：
- 大部分 Tier 2 通过 → 正常交付
- 多项 Tier 2 失败 → 建议优化，但不阻断交付
- 标注存在的 IMPORTANT 问题

### Tier 3: NICE-TO-HAVE（建议优化）

**小石头**：
```text
- 臂长约 0.6-0.9 体高（在合理范围即可）
- 腿长 1-2 段细黑线（功能正常）
- 多人场景 ±10-15% 整体缩放（视觉协调）
```

**老杨**：
```text
- 胡茬细节（两腮侧短茬）
- 眼型细节（适中略垂）
- 肤色深浅（小麦肤范围内）
```

**跨图**：
```text
X0 批内模式语言：一致性（除非用户要求混合）
X3 老杨穿搭肤色：批内协调（同一服装组合）
```

**交付判断**：
- Tier 3 问题不影响交付
- 在交付说明中标注可优化项
- 用户可选择是否返修

## 容忍度量化

### 小石头

| 检查项 | 旧标准 | 新容忍度 |
|--------|--------|----------|
| 臂锚点 | 体侧上 1/3 | 上 1/4 到上 2/5 均可（约 25-40%） |
| 胶囊宽高比 | 1:1.3-1:1.6 | 1:1.2-1:1.7 可接受（±0.1 容忍） |
| 单眼直径 | 体宽 12-18% | 10-20% 可接受（±2% 容忍） |
| 眼距 | 1.2-1.8 眼径 | 1.0-2.0 眼径可接受（视觉协调） |
| 多人缩放 | ±10-15% | ±15-20% 可接受（不同距离感） |

### 老杨

| 检查项 | 旧标准 | 新容忍度 |
|--------|--------|----------|
| 年龄 | 35-40 岁 | 33-42 岁可接受（±2 岁边界容忍） |
| 发长 | 3-5cm | 2.5-6cm 可接受（短直发即可） |
| 眼镜镜片 | 大镜片 | 中等偏大即可（识别为眼镜） |
| 脸型宽度 | 明显宽度 | 不是窄瘦脸即可（避免韩范） |

### 跨图

| 检查项 | 旧标准 | 新容忍度 |
|--------|--------|----------|
| 形体一致 | 批内精确 | ±10% 视觉波动（光线/角度影响） |
| 眼距一致 | 批内精确 | ±5% 视觉波动（透视影响） |
| 年龄一致 | 同一档 | 范围内波动（35-40 岁内均可） |

**风险 1**：质量下降（标准放宽）
- **应对**：Tier 1 保持严格，只是细节降级
- **验证**：对比优化前后生成图质量

**风险 2**：用户期望不匹配
- **应对**：在交付说明中标注 Tier 2/3 问题
- **选择**：用户可选择是否优化

**风险 3**：Agent 混淆分级
- **应对**：文档清晰标注 Tier 等级
- **引导**：QUICK-START 强调 Tier 1 优先
