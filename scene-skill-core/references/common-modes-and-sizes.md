# 尺寸池与图内信息量分级

本文件是所有模式（实物图、手绘图、知识卡、PPT 演讲）共用的尺寸和信息量语言，避免每个模式各说各的。

## 尺寸池

| 出图类型 | 默认尺寸 | 所属模式 |
| --- | --- | --- |
| 正文配图 | 16:9 | 实物图模式 / 手绘图模式 |
| 彩蛋长卷故事图 | 2.6:1–3:1 超横版 | 实物图长卷模式 |
| 图文号知识卡 | 3:4 或 4:5 | 知识卡模式 |
| 手机海报 | 9:16 | 知识卡模式 |
| PPT 演讲页面 | 16:9 | PPT 演讲模式 |

用户可以直接改尺寸；以上只是默认推荐，不是强制锁死。

## 图内信息量分级

对用户说“图内信息量”，不要说内部分级代号：

| 用户侧说法 | 解释 | 适用模式 |
| --- | --- | --- |
| 画面更空 | 只讲一个点，文字很少，留白强 | 实物图 / 手绘图，优先这一档 |
| 讲清楚重点 | 一个主画面 + 几个短标签 | 手绘图（中等复杂度）、轻量知识卡 |
| 内容更完整 | 放步骤、路径、模块、风险或行动 | 知识卡模式 |
| 完整海报 | 信息最多，但仍要保证阅读顺序 | 知识卡模式（课程总览、方法全景等场景），不用于实物图/手绘图 |

实物图模式和手绘图模式只使用「画面更空」「讲清楚重点」两档，不使用「内容更完整」「完整海报」——如果内容确实需要更高信息量，应该切换到知识卡模式，而不是把实物图/手绘图硬塞成信息图。

```text

首选图类型：

出图效果：
推荐原因：

还可以选：
1. [备选]：[效果说明]
2. [备选]：[效果说明]
```
