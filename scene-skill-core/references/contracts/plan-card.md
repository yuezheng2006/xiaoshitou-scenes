# Plan Card · 方案卡

## 用途

方案卡把任务卡转成可执行的视觉方案，是生成前的内容和结构交接单。

## 字段

```text
任务名称：
模式：实物图 / 手绘图 / 知识卡 / PPT / 长卷
比例：
张数或页数：
核心处境或观点：
核心动作：
结构类型或母版：
主物件 / 主结构：
模块数量：
短中文标签：
角色分工：
Persona 分工：
参考资产：
必须保留：
可自由发挥：
失败信号：
```

## 实物图试点的额外字段

```text
母版类型：01-06
母版不变量：
当前内容的至少 3 个变异点：
3 秒读懂句：
硬预算：1 个核心物理动作 / 1 个真实主物件或紧凑主物件组 / 最多 1-2 个小配件 / 最多 4 个短标签
```

## 规则

1. 一张图只承担一个主要观点或物理动作。
2. 母版是质量锚点，不是空间布局模板。
3. 方案卡未完成时，不进入 Render Card。
4. 需要用户选择的内容，应先给方案卡摘要，不要直接生图。
