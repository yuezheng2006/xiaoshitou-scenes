# QA Checklist

## 必过项

### 母版与构图

- 标准模式已锁定 01–06 母版类型；PNG 有则比对，无则用 `physical-master-anchors.md` 骨架。
- 16:9 纯白 `#FFFFFF`；彩蛋长卷近白；留白足够。
- 场景覆盖约宽 60–72%、高 38–50%；3 秒读懂一个物理冲突。
- 无元素清单化、无母版换皮复刻。

### 真实物件现场

- 真实物件清晰、完整、自然，像同一张白色摄影棚照片里的小现场。
- 物件之间光影、透视和轻接触阴影一致；无 UI 截图、App 界面或拼贴边界。
- 主物件必须有真实物件质感和台面关系，不是平面图标或白板线稿。
- 主物件默认现代、轻、干净；不应像老旧库存、复古舞台道具或脏乱工具箱。
- 主物件应符合现代办公 / 数字化协作 / 轻量硬件 / 现代音乐娱乐设备语境；若出现旧行政、旧线下、旧工具隐喻，判不合格。
- 主角色必须和主物件发生接触：挡、推、拉、扛、解、改、接、检查。
- 画面第一眼应像「简笔实物小摆件现场」，不是白板手绘解释图。
- **装饰性测试**：去掉主角色（及出镜的persona），画面道理是否还完全成立？如果完全成立，说明角色是装饰，判不合格（详见 `common-qa-repair.md`）。

### 主角色 IP

- 形体对照 `the active profile main character anchor`，皮肤 `the profile primary color`。
- 白圆眼 + 可选 tiny 瞳孔；**不要** Logo 护目镜眼、不要 anime 眼。
- **2D Flat Lock**：平涂 `the profile primary color` 主色身体、细黑线四肢、平面白圆眼；**禁止** 3D 渐变、高光、体积阴影、软陶/黏土/盲盒/Pixar/充气玩具质感。
- **肢体数量**：每个主角色只能有两只手臂、两条腿；牵引线、绳子、箭头不能和手臂粘连成额外肢体。
- 表情靠 **动作**，汗滴 0–1 个即可。
- **身上** 无 Logo / 品牌名 / 商标标记。
- 2 张及以上：全批同一主角色 + flat 2D。
- 多人同图：同 IP 但不是复制粘贴；每个主角色的体型、倾斜角度、朝向、手脚姿态、站位层次和动作职责至少有 3 项明显不同。

### 作者出镜时（persona）

- 对照 `author-persona-spec.png`：只抓核心识别锚点，不追求面部细节复刻；真实比例长方椭圆脸、圆下巴；大镜片浅灰细框眼镜；短直发 3-5cm；下唇偏厚；眼型适中略垂；**脸侧为主的细微胡茬**；适中稳实身材；米色 T + 深灰运动裤 + 米白运动鞋；无手表。
- persona若与主角色同框：主角色是否仍是严格 flat 2D 的 `the profile primary color` 贴纸风格（平涂主色身体、细黑线肢体、白圆眼），不能因为persona是轻 Q 3D 就被带成 3D 灰石头 / 3D 卵石或任何其他非官方形象；主角色设定见 `the active profile main character anchor`。
- **年龄感**：是否读得出 35+ 成熟但不过熟，不是二十多岁光滑幼态脸，也没有显老疲惫、叔感过重、过多皱纹或白发。
- 是轻 Q 3D 数字玩偶质感（圆润体积、柔和边缘、轻微阴影），不是照片级写实人像，也不是和主角色一样的扁平 2D 纸片人。
- persona占主视觉比例是否超过 30%-35%；是否压过主角色和物件冲突、变成人物专访封面或个人品牌海报。
- persona是否在与主角色或场景发生明确互动（递工具/看反馈/扶物件等），而不是并排站立的背景装饰。
- 若persona处于小比例远景：大框眼镜轮廓、自然短直发轮廓（顶部有一定长度）、接近平直发际线 + 长方椭圆脸 / 圆下巴轮廓、短袖+运动裤穿搭与小麦肤色、高大比例、明确动作六项是否仍可辨认；任一丢失按 `common-persona-routing.md` 小比例最低识别线重生成或返修。

### 合规

- 不出现未请求的公司名、Logo、商标、联系方式或个人隐私。
- 如果用户明确要求真实品牌元素，交付时标注由用户自行确认授权。

## 失败信号

- 不像主角色 ref（Logo 眼、星火体、新 mascot 物种）。
- **主角色 3D 漂移**：主色身体有渐变/高光/体积阴影/软陶/盲盒/Pixar 质感；四肢变 3D 圆管；白圆眼变反光球体；像 3D 公仔而不是 flat 2D 贴纸。
- **肢体错误**：同一侧出现两只手臂、手臂分叉、绳线被画成第三只手、腿数不对，尤其是主角色与 persona 或物件连线互动时。
- 过度细节、商业海报、海外 editorial 风。
- 主角色只站看、不承担动作。
- 同批形体/画法不一致。
- 多人场景里的主角色同高同胖、同角度、同手脚姿势，只是平移复制。
- 身上误印 Logo、品牌名或商标；旁边有箭头指向主角色解释品牌身份。
- 堆很多五官细节，与物理动作脱节。
- **主角色被persona带偏成 3D**：persona出镜时主角色被画成灰色鹅卵石、光滑 3D 卵石、软陶公仔或任何非 `the profile primary color` flat 2D 贴纸形象；主角色必须始终保持严格 flat 2D（平涂主色身体、细黑线肢体、白圆眼），即使persona本身是轻 Q 3D 数字玩偶，两者渲染语言也必须明显不同。
- **persona识别丢失**：未触发却出现作者形象；persona变成照片级写实人像或扁平 2D 纸片人；眼镜画成粗黑框/无框/小圆框（应为大镜片浅灰细框）；头发画成贴皮板寸或蓬松爆炸卷发（应为 3-5cm 自然短直发）；画了手表（应为手腕干净无配饰）；穿搭不是米色 T + 深灰运动裤（和 spec 不一致）；额头画得过宽过满、发际线画成夸张的 V 字尖角/M 型或两侧大幅后退；脸型画成尖下巴/V字脸（应为真实比例长方椭圆、圆下巴）；该有细微胡茬却全净或画成络腮胡；嘴唇画得过薄（应为下唇偏厚）；面容画成柔和娃娃脸，或为了成熟感加出明显皱纹、眼袋、深法令纹、显老疲惫（应为 35+ 成熟但不过熟）；穿成长袖针织衫、正装长裤或紧身牛仔裤（应为短袖+运动裤阳光穿搭）；体型画成纤细白领感或夸张肌肉猛男（应为高大结实略健壮）；实物图模式里误用了手绘线稿风格（应为轻 Q 3D）；小比例远景丢失眼镜/发型/脸型/穿搭/体态比例；作者喧宾夺主超过 30%-35% 或变成头像展示/个人品牌海报。
- 主物件只是平面线稿、图标或纸片符号，没有真实物件质感和台面关系。
- 主物件过旧、复古、笨重或脏乱，导致画面像怀旧海报、舞台道具或库存杂物，而不是现代白底小现场。
- 主物件虽然干净但语义过时，例如旧行政审查、旧线下筛选、传统称重或怀旧设备隐喻，导致画面不符合现代办公 / 数字化协作语境。
- 画面靠箭头和文字解释，像手绘图模式，而不是物理现场。

## 交付判断

1. 第一眼：简、干净。
2. 第二眼：看懂物件与主角色的物理冲突。
3. 标签：共鸣。

不合格 = 重生成，不要交付。

## 收到用户返修意见时

先查 `common-qa-repair.md` 的用户反馈映射表，找到对应返修方向，再按其中的返修输出格式（本轮保留/本轮调整/返修提示词）给出结果，不要凭感觉现改或整套重来。
