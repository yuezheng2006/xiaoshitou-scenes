# 老杨 × 小石头 互动场景库

本文件只服务于**默认 profile 触发 persona 后**的双 IP 互动编排。通用 persona 机制见 `common-persona-routing.md`；老杨分层规范见 profile 下 `persona-author*.md`（入口 `persona-author.md`）。

## 双 IP 分工（默认）

触发老杨 / yuezheng2006 等 persona 词后，默认启用**双 IP 互动**，不是「小石头单主角 + 老杨偶尔露脸」：

```text
老杨 = 主讲 / 拆解 / 批注 / 调度 / 指向结论
小石头 = 执行 Agent：搬卡、贴标签、标风险、递交结果、试错、守门、拉线
```

用户可见语言统一写成「老杨 + 小石头」，不要写内部代称。未触发 persona 时，本文件不适用，仍按单主角色（小石头）流程走。

## 画面权重（按模式）

| 模式 | 老杨 | 小石头 | 场景重心 |
| --- | --- | --- | --- |
| 知识卡 / PPT / 手绘（老杨触发） | 主讲，约 40–55% | 1–3 个执行分工，约 25–40% | 结构 + 互动 |
| 实物图（老杨触发） | 调度 / 旁白 / 递工具，约 25–40% | 核心物理动作，约 30–45% | 物件小现场 |
| 彩蛋长卷（老杨触发） | 逐段旁白 / 递节点卡 | 每节点物理动作 | 人生线物件 |
| 观点金句 / 9:16 海报（老杨触发） | 主讲占主视觉 | 0–1 个执行点缀 | 一句话传播 |

硬性底线：

- 老杨触发后，**每张图都必须有明确的双 IP 互动**，不能只有老杨肖像或只有小石头干活老杨站桩。
- 小石头永远保持 flat 2D（`#f39800` 平涂胶囊），不能因为老杨是 3D 就被带成 3D 公仔。
- 实物图里真实物件仍是第三主角；双 IP 围绕物件协作，不是人物抢镜。

## 六类标准互动场景

### 1. 白板拆解（手绘 / 知识卡 / PPT）

适合：流程、方法论、系统结构、Agent 工作流。

```text
老杨：画结构 / 写批注 / 指向模块 / 圈重点
小石头：拉线 / 搬模块 / 守门 / 把卡片塞进流转槽
```

推荐结构类型：Workflow、系统局部、方法分层（见 `handdrawn-composition-patterns.md`）。

### 2. 知识卡调度（知识卡 / PPT 方法页）

适合：步骤、对比、案例、风险、行动清单、课程总览。

```text
老杨：贴近标题或结论区，主讲 / 指向 / 总结（须正面或 3/4 可见面部识别，禁止背身仅气泡）
小石头 × 1–3：分别在步骤里搬卡、贴标签、标风险、递交反馈、演示失败案例
```

默认 2–3 个小石头分工；纯观点卡可只留 1 个小石头或 0 个。

### 3. 物件现场协作（实物图）

适合：处境、痛点、项目故事、服务现场、工牌 / K 歌道具语境。

```text
真实物件主现场 + 小石头执行物理动作 + 老杨递工具 / 看反馈 / 扶物件 / 写便签
```

推荐读 `physical-object-patterns.md` 选物件类型，再叠加双 IP 动作。

### 4. 大判断 / 金句主讲（知识卡 / PPT / 9:16）

适合：反常识判断、一句话观点、直播切片引流。

```text
老杨：占主视觉，指向核心判断
小石头：0–1 个，做执行点缀（举卡、贴标签、标问号）
```

不是头像海报——必须有指向性动作或和图内文字的结构关系。

### 5. 分镜对比（手绘 / 知识卡）

适合：踩坑复盘、前后变化、失败到修正。

```text
2–4 格：老杨讲解转折；小石头在每格承担具体动作变化
```

用 `handdrawn-composition-patterns.md` 的小漫画分镜或知识卡的分镜对比卡形态。

### 6. 长卷旁白（彩蛋长卷）

适合：个人经历、项目复盘、产品演化。

```text
每个节点：真实物件 + 小石头物理动作 + 老杨递节点卡 / 写旁注 / 指向下一节点
```

长卷里老杨可以小比例出现，但必须满足 `persona-author-identity.md` 的小比例最低识别线。

## 失败信号

- 只有老杨，没有小石头（persona 触发任务）。
- 只有小石头干活，老杨站桩或无动作。
- 老杨变成头像 / 专访封面 / 个人品牌海报。
- 小石头变成装饰，去掉后画面道理仍成立（见 `common-qa-repair.md` 装饰性测试）。
- 知识卡 / 方法论类只有老杨 + 文字，没有小石头执行分工。
- 多个小石头 copy-paste，没有明确执行分工差异。
- 小石头 3D 漂移，或老杨 / 小石头渲染语言混用（实物图里老杨 3D + 小石头必须 flat 2D）。
