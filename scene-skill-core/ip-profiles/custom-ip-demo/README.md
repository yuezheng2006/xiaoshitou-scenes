# Custom IP Demo

这个目录演示自定义 IP 的完整资产链：

```text
用户真实参考图
  → Profile Enrollment
  → 身份锚点确认
  → 模式校准图
  → 正式场景 / 知识卡 / 动作库
```

## 文件说明

- `assets/reference/`：用户提供的参考图，按主身份和变体重命名。
- `assets/examples/`：基于参考图生成的模式示例。
- `profile.md`：Profile Contract 实例与录入状态。
- `character.md`：角色身份锁和 Prompt 槽位摘要。

## 重要边界

这些参考图由用户提供，包含第三方角色 / 品牌元素，仅用于本地流程演示。它们默认属于 `private_assets`，不代表项目拥有公开分发或商业使用授权；提交公开包前应替换为已授权的原创 IP 参考图。
