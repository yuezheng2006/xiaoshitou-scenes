## Identity Lock

```text
User-provided reference is the highest identity anchor.
Keep the same shaggy long ears, rounded plush head/body proportion, large oval eyes, small dark nose, playful toothy smile, short limbs, and compact toy scale.
Use the brown primary reference for identity. The blue reference is a variant, not a replacement identity.
Do not copy pendants, logos, FIFA marks, product text, or third-party branding from the references.
```
