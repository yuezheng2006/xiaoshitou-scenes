# 企业品牌 Logo 使用规范

基于图片提供的 Logo 规范，本文档定义 Logo 在配图场景中的使用边界。

## Logo 版本与色值

### 标准 Logo 版本

**两种配色**（基于视觉需求选择）：

```text
版本 1：黑色版
- 眼睛图形：#000000（纯黑）
- 文字：黑色
- 适用：浅色背景、正式场景

版本 2：橙色版（品牌色）
- 眼睛图形：#F39800（品牌橙）
- 文字：橙色
- 适用：白底配图、活力场景
- 色值：PANTONE 137C
  - CMYK: C0 M51 G94 K0
  - RGB: R243 G152 B0
```

### 横排 Logo（标准形式）

```text
图形 + 中文 + 英文
[眼睛图形] 雷石 THUNDERSTONE

使用场景：
- 一般情况下首选此 Logo
- 无论对内对外，默认使用横排 Logo
```

### 竖排 Logo（空间受限时）

```text
如遇长条形 Logo 空位或特殊位置，可选择竖排 Logo：

[图形]
THUNDERSTONE
雷石 · 手中有麦 身边有你

使用场景：
- 长条形空间
- 特殊版式需求
- 需要配 Slogan 的场景
```

## 使用原则

### 优先级规则

```text
1. 一般情况：首选横排 Logo（标准版）
2. 空间受限：可选竖排 Logo
3. 色值选择：
   - 白底/浅色 → 黑色版或橙色版均可
   - 深色背景 → 橙色版
   - 配合 #F39800 小石头 → 优先黑色版（避免色彩竞争）
```

### 禁止项

```text
❌ 禁止行为：
- 拉伸、变形、旋转 Logo
- 修改 Logo 颜色（仅允许黑色/#F39800 两版本）
- 手绘 Logo 图形（眼睛 swirl）
- 在小石头身上直接印 Logo
- Logo 比例失调（过大或过小）
- 低分辨率模糊 Logo
```

## 工牌使用（雷石员工场景）

### 工牌规格

```text
载体：独立挂绳卡
Logo 位置：工牌牌面（1:1 使用授权资产）
工牌佩戴：小石头独立佩戴（挂绳）
```

### 工牌场景

**触发条件**：
- 用户明确说「雷石员工」「戴工牌」「研发」「AI native」

**默认行为**：
- 小石头：可戴工牌
- 老杨（persona）：默认不戴工牌，除非用户明确要求

**禁止**：
- 工牌写姓名/岗位/流程文字（Logo only）
- 工牌比例过大（喧宾夺主）
- 非雷石员工场景带任何工牌

### 工牌比例参考

```text
小石头体高 = H
工牌高度 ≈ 0.15H - 0.20H（体高的 15-20%）
工牌宽度 ≈ 工牌高度 × 1.5（标准证卡比例）

示例：
- 小石头 200px 高 → 工牌约 30-40px 高
- 工牌不应超过小石头体宽
```

## 道具库（雷石业务场景）

### 允许的业务道具

```text
✅ 可用道具（含 Logo 元素）：
- 曲库更新卡（音乐内容）
- 设备平台卡（车载/家庭/线下三端）
- 权限牌（AI 平台权限）
- 会员反馈卡（用户反馈）
- 系统控制台（抽象化界面）
```

### 道具使用原则

```text
原则：
- Logo 可出现在道具/卡片上（而非小石头身上）
- 比例适中，不抢主角
- 道具服务于叙事（承载信息/任务）
- 避免纯品牌展示（不做成广告海报）
```

## 场景示例

### ✅ 正确用法

**示例 1：雷石员工协作**
```text
场景：Agent 工作流设计
- 小石头戴工牌（独立挂绳卡，Logo 在牌面）
- 工牌比例：约体高 18%
- 背景：白底 + 抽象控制台
- Logo 版本：黑色横排
```

**示例 2：系统架构图**
```text
场景：车载 K 歌平台架构
- 道具：设备平台卡（含 Logo）
- 小石头：递卡、指向、标注
- Logo 版本：橙色横排（与小石头区分）
```

**示例 3：通用配图（无 Logo）**
```text
场景：通用方法论配图
- 无工牌、无 Logo 道具
- 纯小石头 + 抽象物件
- 适用于非特定业务场景
```

### ❌ 错误用法

**错误 1：Logo 直接印在小石头身上**
```text
❌ 小石头胶囊体上出现 Logo 图形或文字
✅ Logo 只在独立工牌/道具上
```

**错误 2：Logo 变形**
```text
❌ 拉伸成瘦长 Logo、压扁 Logo、倾斜 Logo
✅ 保持原始比例，1:1 使用授权资产
```

**错误 3：非业务场景误用工牌**
```text
❌ 通用方法论配图 + 工牌
❌ 个人成长话题 + 工牌
✅ 工牌仅用于明确的「雷石员工」「AI native」场景
```

**错误 4：Logo 喧宾夺主**
```text
❌ Logo 比小石头还大
❌ 整个画面被 Logo 占据（变成广告）
✅ Logo 作为场景元素，不抢主角
```

## 质量检查

### Logo QA 清单

生成后检查：

```text
[ ] Logo 使用了授权版本（黑色或橙色 #F39800）
[ ] Logo 比例正确，未变形
[ ] Logo 在工牌/道具上，未直接印在小石头身上
[ ] 工牌比例适中（约体高 15-20%）
[ ] Logo 清晰可辨，未模糊
[ ] 场景匹配：雷石员工场景才出现工牌
[ ] 非业务场景无误用 Logo
```

### 修复建议

| 问题 | 修复方案 |
|------|----------|
| Logo 变形 | 重新贴入 1:1 授权资产 |
| Logo 印在小石头上 | 删除，改为独立工牌 |
| 工牌过大 | 缩小到体高 15-20% |
| 非场景误用 | 删除工牌，改为通用配图 |
| Logo 模糊 | 使用高清矢量资产 |
| 手绘 swirl 不准 | 禁止手绘，必须贴图 |

## Prompt 引导片段

### 工牌场景 Prompt

```text
Thunderstone employee scene: Little Stone wears a Thunderstone lanyard badge (independent card, logo on card surface only, NOT printed on orange body). Badge size ~15-20% of body height. Logo: black or orange #F39800 horizontal version [eye symbol] 雷石 THUNDERSTONE. Clean proportions, 1:1 logo ratio, no distortion.
```

### Logo 禁止项 Negative

```text
Negative: logo directly printed on Little Stone orange body, logo text as body pattern, oversized badge, distorted/stretched/rotated logo, hand-drawn swirl logo, blurred logo, badge with name/title text, generic badge in non-employee scene.
```
