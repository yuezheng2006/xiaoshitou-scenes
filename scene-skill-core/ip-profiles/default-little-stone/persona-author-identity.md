## 识别优先级（从高到低）

1. **大镜片浅灰/透明细框方形眼镜**（第一识别特征；不是粗黑框、小圆框、无框）
2. **下唇明显比上唇更厚**（第二识别特征）
3. **米色/浅卡其短袖 T + 深灰束脚运动裤 + 米白/奶油色运动鞋**（第三识别特征）
4. **适中偏小、略垂、沉稳的眼睛**（第四识别特征；不要大眼/动漫眼）
5. **脸侧与下颌线极淡短茬**（第五识别特征；不是络腮胡）

## 渲染语言（随模式，识别锚点不变）

- **实物图 / 长卷**：轻 Q 3D 数字玩偶 / 软黏土感；圆润体积、柔和边缘；不要照片级皮肤与布光。
- **手绘 / 知识卡 / PPT**：扁平黑线人物；不要 3D 体积与阴影；锚点仍对齐 spec 图。

手绘版额外要求：不画法令纹、抬头纹、眼袋；成熟感靠脸型、眉、眼型、厚下唇、大眼镜+短直发组合。

老杨占画面约 15–20% 或更小时（参考 `author-persona-actions.png` 远景样例），以下六项缺任一项即判定跑偏：

2. 短直发 3–5cm 轮廓仍可辨认
