# PPT 演讲模式 · 示例成套页面

这里放的是**可以直接用于直播/分享的 PPT 页面图**，不是「展示 Skill 有 PPT 能力」的说明图。

示例主题：《AI 时代，内容怎么配图才有效》（8 页）

| 文件 | 页面类型 | 沟通任务 |
| --- | --- | --- |
| `01-cover.png` | 封面页 | 定主题、建立分享气质 |
| `02-big-judgment.png` | 大判断页 | 好配图服务读者，不是服务作者 |
| `03-pain-points.png` | 痛点页 | 三个创作者常见困境 |
| `04-scene.png` | 场景页 | 从 3000 字长文里只留 3 个共鸣点 |
| `05-method.png` | 方法页 | 配图三步：抽节点 → 选隐喻 → 留 3 标签 |
| `06-case.png` | 案例页 | 一个处境，一张图 |
| `07-timeline.png` | 时间线页 | 试 → 改 → 留一版 |
| `08-closing.png` | 结束页 | 今晚只试一张图 |

出图前必须先出导演规划卡，规则见 `scene-skill-core/references/ppt-presentation-mode.md`。

布局参考 [ip-diagram-creator ppt-mode](https://github.com/haloshin/ip-diagram-creator/tree/main/assets/examples/ppt-mode)——他们的示例同样是**成套演讲页**，不是功能说明书。
