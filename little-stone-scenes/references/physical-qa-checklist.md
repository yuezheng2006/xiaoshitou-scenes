# QA Checklist

## 必过项

### 母版与构图

- 标准模式已锁定 01–06 母版类型；PNG 有则比对，无则用 `physical-master-anchors.md` 骨架。
- 16:9 纯白 `#FFFFFF`；彩蛋长卷近白；留白足够。
- 场景覆盖约宽 60–72%、高 38–50%；3 秒读懂一个物理冲突。
- 无元素清单化、无母版换皮复刻。

### 真实物件现场

- 真实物件清晰、完整、自然，像同一张白色摄影棚照片里的小现场。
- 物件之间光影、透视和轻接触阴影一致；无 UI 截图、App 界面或拼贴边界。
- 主物件必须有真实物件质感和台面关系，不是平面图标或白板线稿。
- 小石头必须和主物件发生接触：挡、推、拉、扛、解、改、接、检查。
- 画面第一眼应像「简笔实物小摆件现场」，不是白板手绘解释图。
- **装饰性测试**：去掉小石头（及出镜的老杨），画面道理是否还完全成立？如果完全成立，说明角色是装饰，判不合格（详见 `common-qa-repair.md`）。

### 小石头 IP

- 形体对照 `little-stone-ref.png`，皮肤 `#f39200`。
- 白圆眼 + 可选 tiny 瞳孔；**不要** Logo 护目镜眼、不要 anime 眼。
- **2D Flat Lock**：平涂 `#f39200` 橙身、细黑线四肢、平面白圆眼；**禁止** 3D 渐变、高光、体积阴影、软陶/黏土/盲盒/Pixar/充气玩具质感。
- 表情靠 **动作**，汗滴 0–1 个即可。
- **身上** 无 Logo / 品牌名 / 商标标记。
- 2 张及以上：全批同一小石头 + flat 2D。
- 多人同图：同 IP 但不是复制粘贴；每个小石头的体型、倾斜角度、朝向、手脚姿态、站位层次和动作职责至少有 3 项明显不同。

### 作者出镜时（老杨）

- 对照 `author-persona-spec.png`：浅灰/透明细框眼镜、自然短直发（3-5cm 毛寸感，带一点自然碎发质感，不夸张不蓬松）、接近平直只有轻微柔和下探的发际线（不是夸张尖角，额头本身不宽）、脸型上宽下窄下巴略尖、嘴唇偏厚唇形清晰、面容硬朗、健康小麦肤色、高大结实略健壮体态（183cm/85kg 比例感，肩宽背厚有克制运动感）、短袖+运动裤穿搭是否都在。
- 老杨若与小石头同框：小石头是否仍是严格 flat 2D 的 `#f39200` 贴纸风格（平涂橙身、细黑线肢体、白圆眼），不能因为老杨是轻 Q 3D 就被带成 3D 灰石头 / 3D 卵石或任何其他非官方形象；小石头设定见 `little-stone-ref.png`。
- **年龄感**：是否读得出真实的 35+ 偏近 40 的成熟感，不是二十多岁光滑幼态脸，也没有显老疲惫或过多皱纹白发。
- 是轻 Q 3D 数字玩偶质感（圆润体积、柔和边缘、轻微阴影），不是照片级写实人像，也不是和小石头一样的扁平 2D 纸片人。
- 老杨占主视觉比例是否超过 30%-35%；是否压过小石头和物件冲突、变成人物专访封面或个人品牌海报。
- 老杨是否在与小石头或场景发生明确互动（递工具/看反馈/扶物件等），而不是并排站立的背景装饰。
- 若老杨处于小比例远景：细框眼镜轮廓、自然短直发贴头轮廓、接近平直的发际线/上宽下窄脸型轮廓、短袖+运动裤穿搭与小麦肤色、高大比例、明确动作六项是否仍可辨认；任一丢失按 `common-author-persona.md` 小比例最低识别线重生成或返修。

### 合规

- 不出现未请求的公司名、Logo、商标、联系方式或个人隐私。
- 如果用户明确要求真实品牌元素，交付时标注由用户自行确认授权。

## 失败信号

- 不像小石头 ref（Logo 眼、星火体、新 mascot 物种）。
- **小石头 3D 漂移**：橙身有渐变/高光/体积阴影/软陶/盲盒/Pixar 质感；四肢变 3D 圆管；白圆眼变反光球体；像 3D 公仔而不是 flat 2D 贴纸。
- 过度细节、商业海报、海外 editorial 风。
- 小石头只站看、不承担动作。
- 同批形体/画法不一致。
- 多人场景里的小石头同高同胖、同角度、同手脚姿势，只是平移复制。
- 身上误印 Logo、品牌名或商标；旁边有箭头指向小石头解释品牌身份。
- 堆很多五官细节，与物理动作脱节。
- **小石头被老杨带偏成 3D**：老杨出镜时小石头被画成灰色鹅卵石、光滑 3D 卵石、软陶公仔或任何非 `#f39200` flat 2D 贴纸形象；小石头必须始终保持严格 flat 2D（平涂橙身、细黑线肢体、白圆眼），即使老杨本身是轻 Q 3D 数字玩偶，两者渲染语言也必须明显不同。
- **老杨识别丢失**：未触发却出现作者形象；老杨变成照片级写实人像或扁平 2D 纸片人；眼镜画成粗黑框（应为浅灰/透明细框）；头发画成蓬松爆炸卷发（应为自然短直发，3-5cm 毛寸感，不需要死贴头皮但不能蓬松打卷）；额头画得过宽过满、发际线画成夸张的 V 字尖角/M 型或两侧大幅后退（应为接近平直、只有轻微柔和下探的自然发际线，额头本身不宽）；脸型画成方脸宽下巴（应为上宽下窄、下巴略尖）；嘴唇画得过薄或没有唇形细节（应为嘴唇偏厚、唇形清晰）；皮肤画得过白（应为健康小麦色）；面容画成柔和娃娃脸（应为硬朗轮廓）；画成二十多岁光滑幼态脸或过度显老疲惫（应为真实 35+ 偏近 40 的成熟感）；穿成长袖针织衫、正装长裤或紧身牛仔裤（应为短袖+运动裤阳光穿搭）；体型画成纤细白领感或夸张肌肉猛男（应为高大结实略健壮）；实物图模式里误用了手绘线稿风格（应为轻 Q 3D）；小比例远景丢失眼镜/发型/脸型/穿搭/体态比例；作者喧宾夺主超过 30%-35% 或变成头像展示/个人品牌海报。
- 主物件只是平面线稿、图标或纸片符号，没有真实物件质感和台面关系。
- 画面靠箭头和文字解释，像手绘图模式，而不是物理现场。

## 交付判断

1. 第一眼：简、干净。
2. 第二眼：看懂物件与小石头的物理冲突。
3. 标签：共鸣。

不合格 = 重生成，不要交付。

## 收到用户返修意见时

先查 `common-qa-repair.md` 的用户反馈映射表，找到对应返修方向，再按其中的返修输出格式（本轮保留/本轮调整/返修 prompt）给出结果，不要凭感觉现改或整套重来。
