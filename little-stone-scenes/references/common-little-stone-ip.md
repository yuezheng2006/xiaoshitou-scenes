# 小石头 IP

## 核心定位

**小石头 (Little Stone)** — flat 2D 简笔正文配图角色：物理动作、短标签、大留白。

```text
形体：圆角胶囊 / 竖椭圆，#f39200 橙实心体
眼睛：白色小圆 + 可选 tiny 瞳孔（不要 Logo 护目镜、不要 anime 大眼）
肢体：细黑线；表情靠动作，汗滴 0–1 个即可
身上：禁止任何 Logo / 品牌名 / 商标印花
```

**角色锚点（唯一）**：`assets/brand/little-stone-ref.png`

## 品牌血脉（全篇唯一点明处）

小石头的「特色」内化在 IP 本体里，不靠正文反复说明：

- **品牌橙 `#f39200`**：小石头的皮肤色即雷石品牌主色，是它最强的识别符号。
- **命名血脉**：`小石头 (Little Stone)` 的名字源自「雷石」的「石」，提取其稳、钝、倔的气质，做成一个独立的原创配图角色。

约定：除本节、`NOTICE.md` 合规声明，以及用户明确触发的可选品牌 Logo 资产外，正文、示例和提示词**一律不展开雷石公司业务细节**，避免敏感信息外泄。品牌感只通过 `#f39200` 橙身、胶囊体和物理动作自然传达，而不是反复书写公司名。

## 是谁

略萌、认真、略笨、有点倔——靠 **物理动作** 表达，不堆五官。

## 默认形象

### 身体

- 品牌橙 `#f39200` 实心体。
- 轮廓：简洁 **圆角胶囊 / 竖椭圆**，高宽比约 **1.2:1 ~ 1.5:1**。
- 同图可多小人、略拉长/略压扁——**仍是小石头**，不要变成另一种 mascot。
- **全身纯色 #f39200**，禁止 Logo / 品牌名 / 任何商标。

### 多人 / 多小石头

当用户要求多人、多个 Agent、团队协作、接力、评审会、结对、三方对齐等场景时，可以画 2-4 个小石头。它们应是同一个 IP 家族里的不同个体，**不要像复制粘贴**：

- 必须共享：`#f39200` 橙身、白圆眼、细线肢体、flat 2D 简笔、无 Logo。
- 必须拉开：身高、胖瘦、倾斜角度、朝向、手脚姿态、站位层次、正在做的动作。
- 每个小石头要有不同职责，例如递卡、接卡、扶线、贴标签、看反馈、按住物件。
- 体型变化建议明显但克制：一个高一点瘦一点、一个矮一点胖一点、一个前倾、一个后仰；仍保留胶囊 / 竖椭圆识别。
- 禁止只复制同一个小石头并平移位置；第一眼像 clone stamp / copy-paste 即失败。

### 眼睛

- **两个白色小圆眼**，可带 **tiny 黑色瞳孔点**；不要 Logo 叠眼、不要大护目镜、不要 anime 眼。
- 表情优先靠 **动作、倾斜、汗滴（0–1 个）**；眉毛 **可选**，最多 **一条** 简单短线。

### 肢体

- 细线胳膊、细线腿；认真做物理动作。

## 同类场景一致性

一次生成 **2 张及以上** 时，Character Lock：

| 全批一致 | 可变 |
| --- | --- |
| `#f39200` 橙身 | 姿势 / 动作 |
| 胶囊轮廓（对照 ref） | 0–1 汗滴 |
| 白色小圆眼 + 可选 tiny 瞳孔 | 场景道具 |
| flat 2D 简笔 | 标签 |
| 细线肢体 | |

**QA**：对照 `little-stone-ref.png`——同一 **小石头**，不是新物种。多人同图还要检查：每个小石头的体型、姿态、朝向和动作分工是否明显不同；如果只是同一个形体复制平移，不合格。

## 禁止

- 脱离 ref 形体（Logo 导出体、麦克风人、桌游风等 unrelated mascot）。
- 小石头本体变成复杂眉眼、3D 写实角色或海外精致插画。实物图里的主物件可以有真实物件质感，但小石头仍保持 flat 2D 简笔。
- 小石头 **身上或旁** 出现 Logo / 品牌名 / 商标引导箭头。
- 同批变成不同 mascot 或不同画法。

## 必须承担核心动作

删掉小石头后隐喻仍成立 = 失败。
